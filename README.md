# test-rabbitmq

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run start:all
```

To up the RabbitMq server:
```bash
docker compose up
```

To access management portal:

[Management portal](http://127.0.0.1:15672)

- user: guest
- password: guest
